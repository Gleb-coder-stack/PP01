# Система учёта мероприятий АНО МЦИ «ШАГ НА ВСТРЕЧУ»

## О проекте

Данный проект разработан в рамках производственной практики ПП01 студентом **Феоктистовым Глебом Юрьевичем**.

Система предназначена для автоматизации учёта мероприятий инклюзивной танцевальной организации **АНО МЦИ «ШАГ НА ВСТРЕЧУ»**.

### Основные возможности

- 📅 **Календарь мероприятий** — визуальное отображение всех событий по датам с цветовой индикацией статусов
- 📋 **Управление мероприятиями** — добавление, редактирование, удаление мероприятий (дата, время, место, тип, описание)
- 👥 **Управление участниками** — добавление, редактирование, удаление участников с указанием категории (ребёнок, взрослый, волонтёр, особенный гость)
- ✍️ **Запись на мероприятия** — запись участников на мероприятия через удобный интерфейс (без авторизации), возможность быстрого добавления нового участника
- 📊 **Формирование отчётов** — экспорт журнала мероприятий и статистики в PDF-формат
- 📝 **Логирование изменений** — автоматическая фиксация всех операций в таблице `logs`
